## 视频教程

:::info 1 分钟通关 OCS 网课助手 移动端安装教程！

自动答题、管理脚本等教程请看 脚本教程：[https://docs.ocsjs.com/docs/script#open-answerer](/docs/script#open-answerer)

:::

:::warning 注意！

**切后台**挂网课，进行其他操作，**刷抖音玩游**戏，需要使用**手机分屏**功能，吧浏览器分小屏出来，否则会**影响脚本运行**。

或者使用**平板、备用手机**挂着刷课。

:::
