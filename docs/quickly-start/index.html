OCS - 电脑桌面软件（✨ 推荐）​

适合零基础小白

保姆式一键刷课、多账号刷课、软件辅助、浏览器多开/分身、网课账号管理、自动登录等功能

🔗 使用教程：https://docs.ocsjs.com/docs/app

OCS - 移动端教程​

适合手机、ipad 平板等移动端设备

🔗 使用教程：https://docs.ocsjs.com/docs/mobile

OCS - 网课脚本​

适合电脑浏览器安装了脚本管理器（如：脚本猫、Tampermonkey）的用户

用于自动学习 超星学习通、知道智慧树、职教云、智慧职教 等多个网课

🔗 使用教程：https://docs.ocsjs.com/docs/script
