使用须知​

不要使用 QQ 、360 等浏览器（不兼容） ！
目前兼容的浏览器（点击展开）：
Edge 微软浏览器 https://www.microsoft.com/zh-cn/edge/download
Chrome 谷歌浏览器 https://www.google.cn/chrome/index.html

第一步 安装 脚本管理器​

以下两个脚本管理器只需安装其中一个即可 ！
点击展开

🎉 脚本猫是 一之哥 开发的国产脚本管理器哦，多多支持~

脚本猫 安装-图文教程：
  • 脚本猫链接： https://docs.scriptcat.org

点击获取即可安装。安装后，即可查看第二步。

Tampermonkey（篡改猴/有些人也叫他油猴，但油猴其实是 GreaseMonkey ， 最早出现的脚本管理器）

Tampermonkey 安装-图文教程：
  • Tampermonkey 链接： https://www.tampermonkey.net/

点击获取即可安装。安装后，即可查看第二步。

第二步 安装脚本​

关于 2025 年多个刷课问题公示 ： 点击查看
在使用之前，请关闭其他脚本，避免脚本之间发生冲突 ！如果页面出现多个 OCS 脚本，请参考下方的 删除脚本教程 ，确保只有一个脚本
请先查看 第一步安装脚本管理器 ，否则将无法安装脚本到管理器中进行使用，而是下载脚本文件
在安装页面点 安装/更新 按钮后，页面会 自动关闭 ，这是 正常现象
安装来源链接说明
✨推荐 官方安装链接🔗 前往安装/更新OCS 官方脚本链接
脚本猫脚本站🔗 前往安装/更新脚本猫旗下的脚本托管网站
greasyfork（油叉，国内暂时无法访问）🔗 前往安装/更新油叉是当下比较热门的的脚本托管网站
github（需科学上网）🔗 前往安装/更新由 Github Action 自动打包生成（最新）

更新脚本教程

关闭脚本教程

注意每个脚本管理器的页面是不一样的，但是关闭逻辑大致相同。

  1. 打开右上角的脚本管理器，点击脚本管理。

  1. 然后进入脚本列表，找到左边开关按钮，点击即可关闭

删除脚本教程

注意每个脚本管理器的页面是不一样的，但是删除逻辑大致相同。

  1. 打开右上角的脚本管理器，点击脚本管理。

  1. 然后进入脚本列表，找到右边删除按钮，点击即可删除
